# StreetARTists

## Mobile-First Application:
This application is designed specifically for mobile devices, ensuring an optimal experience for users on smartphones.

## General
### Project Goal:
To create a web space for every street artist that allows them to track their income and have a place on the web where they can showcase their masterpieces.

### What is the vision?
To help young artists express themselves through modern technology and give them the opportunity to be reachable worldwide.

## Installation

 ```

```

2. Navigate to the folder:
```
cd your-folder-name
```

## Usage
Open the index.html file in your preferred mobile web browser to view the project. This application is designed for mobile devices, so it is recommended to use a smartphone or tablet for the best experience.

Feel free to customize the content as per your specific requirements and details.
